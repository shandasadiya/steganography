Welcome to ShadowVault

A simple, lightweight educational platform for hiding secret messages inside images, generating private cryptographic keys, and protecting documentation.

Image Steganography

Hide a secret text message inside an image file using your browser canvas. The visual properties of the image remain identical.

Private Key Generator

Generate secure, random cryptographic keys locally to learn how secure encryption seeding operates.

Document Protection

Encrypt raw text files locally using secure, client-side browser logic without exposing information to servers.

Educational Notice: This project is designed strictly for educational exploration. Do not rely on client-side browser implementations alone to secure enterprise or highly critical personal infrastructure.
